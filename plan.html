🎯 ما هذا المشروع؟

زحمة الشحن لعبة ألغاز عربية تُلعب في المتصفح مباشرة. شاحنات محشورة في مستودع، كل واحدة تحمل مادة، وعليك ترتيب خروجها وتوصيل الطلبات قبل أن ينفد صبر الزبائن. لا build، لا مكتبات، لا backend — مجرد HTML + CSS + Canvas/JS، وواجهتها بالكامل عربية (RTL).

🔁 حلقة اللعب

  1. الشاحنة تخرج فقط إذا كان مسارها باتجاه وجهتها حتى حافة المستودع فاضياً — هنا تكمن الزحمة.
  2. الضغط على شاحنة قابلة للوصول ينقلها عبر طريق دائري إلى رصيف تحميل فاضٍ.
  3. في الرصيف تُحمّل وحدة واحدة كل مرة كلما طابقت مادتُها طلباً قائماً.
  4. الشاحنات الكبيرة (حجم ٢) تشغل خليتين وتحمل وحدتين — تسدّ أكثر وتنتظر أطول.
  5. الطلب المنتهي يكلّف قلباً. صفر قلوب = خسارة. كل الشاحنات وصلت = فوز.

⚖️ الثوابت الحرجة (لا تُكسر)

fixDeadlock() يضمن قابلية الحل دائماً — يُستدعى بعد التجهيز والتحميل وتوليد الطلبات.
الطلب يُولَّد من المخزون المتبقّي (availableSupply) فيبقى دائماً قابلاً للتحقيق.
اللاعب يخسر فقط عبر انتهاء الطلب — لا يجوز أبداً أن تصبح اللوحة غير قابلة للحل.
