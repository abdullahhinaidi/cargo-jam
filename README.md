# 🚚 زحمة الشحن — Cargo Jam

لعبة ألغاز شحن ولوجستيات تعمل في المتصفح مباشرة — بدون تثبيت وبدون خادم.

**العب الآن:** https://abdullahhinaidi.github.io/cargo-jam/

## طريقة اللعب

مستودع مزدحم بالشاحنات، وكل شاحنة تحمل مادة معيّنة (🪵 خشب، 🛢️ نفط، 🍎 غذاء، 🔩 حديد، 📦 بضائع، 💧 ماء).

- فوق تظهر **طلبات الزبائن** — كل طلب يبي مادة بكمية معيّنة وله **مؤقّت صبر**.
- اضغط على الشاحنة المطلوبة (الإطار الأخضر) فتخرج من الزحمة، تلفّ في الطريق، وتدخل **رصيف التحميل**.
- الشاحنة تتحمّل وحدة تلو الأخرى حسب الطلب، ولمّا تكتمل حمولتها تنطلق.
- **الشاحنة الكبيرة** تحمل وحدتين وتسدّ خانتين — تحتاج تخطيط أكثر.
- الشاحنة ما تقدر تخرج إلا إذا كان طريقها للحافة فاضي — **هنا الزحمة**.
- لو نفد صبر طلب تخسر قلباً. تخلص القلوب = تخسر.

**التحدي:** توازن بين العرض (الشاحنات اللي تقدر توصلها في الزحمة) والطلب المتغيّر، مع عدد مواقف تحميل محدود.

## المميزات

- **٣٠ مرحلة** بمنحنى صعوبة متدرّج، والمواد تُدخَل تدريجياً
- نظام **نجوم (١–٣)** وحفظ التقدّم تلقائياً على الجهاز
- قائمة رئيسية، اختيار مراحل، إعدادات، وإيقاف مؤقّت
- مؤثرات صوتية وموسيقى خلفية (يمكن إطفاؤها)
- تعمل على الجوال والآيباد واللابتوب

## التشغيل محلياً

افتح `index.html` مباشرة، أو شغّل خادماً محلياً:

```bash
python -m http.server 8777
# ثم افتح http://localhost:8777
```

## البنية

| الملف | الوظيفة |
|------|---------|
| `index.html` | الشاشات (قائمة، مراحل، إعدادات، لعب) |
| `style.css` | التصميم والتجاوب مع الأحجام |
| `levels.js` | مولّد الـ٣٠ مرحلة ومنحنى الصعوبة |
| `game.js` | المحرّك، الرسم، الحفظ، الصوت |

`car-version/` و `bus-version/` نسخ سابقة من تطوّر الفكرة (لعبة سيارات ولعبة باصات).
